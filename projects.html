Наши разработки

BACS

BACS - Программный комплекс, предназначенный для проведения соревнований по спортивному программированию. Также может использоваться в учебном процессе для проверки лабораторных работ, связанных с программированием. Подробнее →

AcmMuseum

AcmMuseum - сайт виртуального музея ACM олимпиад. В рамках данного проекта была разработана система для созданиая виртуального пространства. Подробнее →

WebKafedra

Проект WebKafedra - проект по написанию web-интерфейса, позволяющего преподавателям кафедры удаленно выкладывать методические материалы и лабораторные материалы для студентов, а студентам скачивать эти материалы. Подробнее →

Портал кафедры

Портал разработан студентами для студентов и преподавателей. Главная цель сайта — предоставление актуальной информации о кафедре, преподавателях и предметах. Подробнее →
