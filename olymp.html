Олимпиады

Олимпиадами на нашей кафедре интересуются с 1996 года. За это время было подготовлено много спортсменов, получены награды из Праги(серебро, 2004 год), Шанхай(бронза) и золото из Канады.

История подобных соревнований насчитывает уже два десятка лет. По правилам команде, состоящей из трех участников, предоставляется один компьютер и предлагается в течении 5 часов решить максимальное число из предложенных задач. Побеждает команда, решившая наибольшее число задач. При равенстве числа сданных задач учитывается ещё и время. Итоги турнира подводятся в реальном времени. Задачи отправляются в тестирующую систему, где они немедленно проверяются. Обновление таблицы результатов тоже происходит автоматически.

Если вам интересно спортивное программирование и вы хотите прославить себя или родной университет на чемпионатах по программированию - у вас есть такой шанс.

Достижения

BACS - Программный комплекс, предназначенный для проведения соревнований по спортивному программированию. Также может использоваться в учебном процессе для проверки лабораторных работ, связанных с программированием.
